Disadvantages with virtualization techniques:-
	1.The guest OS is not as effective as Base OS
	2.We are wasting the resources
	3.If we need more resources, we need to invest more money.
	4.Application portability is not guaranteed.
	5.Manually we r allocating all the resources VM's

To overcome all the challenges we use Containerization.
