Linux Permissions:
	types - rwx
	r - read
	w - write
	x - execute

	groups -
	current user(u) | group(g) | other(o)

	commands - 
		chmod - change permissions
		chgrp - change group of file
		chown - change owner of file

	read-4
	write-2
	execute-1


Vertical scaling - Increase existing servers memory/resources (scale up)
Horizontal scaling - Increase number of servers (scale out)

scale up - increase needed resources when traffic increases
scale down - when traffic is coming down, we have to remove the additional servers, whatever we added based on the traffic.
